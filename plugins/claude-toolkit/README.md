# Claude Toolkit

Skills for creating Claude Code plugins, skills, marketplaces, and hooks.

## Skills

| Skill | Description |
|-------|-------------|
| `claude-skills` | Guide for creating Agent Skills - SKILL.md files, directory structure, progressive disclosure, best practices |

## Installation

```shell
/plugin marketplace add maledorak/maledorak-marketplace
/plugin install claude-toolkit@maledorak-marketplace
```
