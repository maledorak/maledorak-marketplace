# Common Development Skills

Essential development skills for Claude Code workflows.

## Included Skills

### git-commit
Guide for creating semantic, version-aware commit messages following Conventional Commits 1.0.0 specification.

**Use when:** Making git commits, writing commit messages, reviewing commit history

**Key features:**
- Type taxonomy (feat/fix/docs/refactor/perf/test/chore)
- Breaking change indicators
- SemVer correlation
- Comprehensive examples
